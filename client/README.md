# Connex Client - React Frontend

This is the React frontend for the Connex MERN stack application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file (already created) with:
```
REACT_APP_API_URL=http://localhost:5000/api
```

3. Start the development server:
```bash
npm start
```

The app will open at http://localhost:3000

## Features

- Sample page that fetches users from the backend API
- Tailwind CSS for styling
- Ready to expand with full application features

## API Connection

The frontend is configured to connect to the backend API at `http://localhost:5000/api`.

Make sure the backend server is running before starting the frontend.
